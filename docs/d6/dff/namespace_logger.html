A print statement that overrides the default print function.

It also logs the parameter str to the log file from Constants.py

Parameters
str(str) - The string to print and log.
end(str) - The string to end the print with (defaults to newline).
flush(bool) - Whether to flush the string immediately or not.
onlyLog(bool) - True: Only log str to the log file. False: Log str to both terminal and the log file.
logLevel(str) - The severity level to log str as.
