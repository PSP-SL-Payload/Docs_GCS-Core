Detailed Description

This class represents a background reading thread that the SerialInterface class creates to read lines from the serial port and append them to a list.

None SerialInterface.ReadingThread.run (   self)

Method that is called when ReadingThread.start() is called.

This runs in the background forever until stop() is called. It continuosly reads from the serial port and fills up a serial buffer inside the SerialInterface class instance.
